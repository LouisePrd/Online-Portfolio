Project type : Website
Date : 2025
Context : Solo student project at IMAC
Technologies used : Html, CSS, JS + API LoL

Description :
If you've stumbled upon League of Legends and think you know everything there is to know about the game, you've come to the right place. Use all those hours spent playing to learn about the characters, their skills, and how to play to challenge yourself.
