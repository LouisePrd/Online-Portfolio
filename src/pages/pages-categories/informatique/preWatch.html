Interactive 3D Visualization @preWatch

Developing a next-generation 3D web service using Gaussian Splatting technology.

Project type : Web Service & 3D Engine
Date : 2024 - 2025
Context : Internship & Freelance
Technologies used : PlayCanvas (WebGL), Next.js, GitHub Actions, Gaussian Splatting

Description :
During my time at preWatch, first as an intern and then as a freelancer, I contributed to the development of a professional 3D visualization platform based on Gaussian Splatting data.

My main focus was developing a dynamic user interface using the PlayCanvas engine. This interface allows users to interact deeply with 3D scenes directly in the browser, including precision tools for measurements, model integration, and scene customization.

To optimize our development cycle, I implemented a continuous integration workflow using GitHub Actions, automating the synchronization between our codebase and PlayCanvas projects.

More recently, I collaborated with the lead developer on the database architecture to integrate our WebGL work into a comprehensive Next.js Dashboard. This evolution aims to provide a full management experience, covering project settings, team collaboration, and real-time modifications.
