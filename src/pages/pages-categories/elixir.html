Elixir

Site web dédié à la découverte et à la préparation de cocktails à l'aide d'une API publique.

Type de projet : Site web
Date de réalisation : 2023
Contexte : Projet étudiant à l'IMAC
Technologies utilisées : Html, CSS, JS + API thecocktaildb

Description :
Plongez tête la première dans l'univers des cocktails. Grâce à Elixir, vous trouverez rapidement la recette qui vous permettra de réaliser toutes vos envies et de faire pétiller vos papilles, ou de réveiller le mixologue qui sommeille en vous pour essayer de deviner les ingrédients... A la vôtre !
